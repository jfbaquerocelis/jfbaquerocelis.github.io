Mis Herramientas

Y como buen desarrollador web, necesito herramientas que me ayuden a ayudarte. Estas son las que uso actualmente.

  • Frontend
  • Backend

HTML5

Si de maquetar se habla, HTML5 es la herramienta correcta. Su sitio web lo necesitará.

CSS3

En diseño, CSS3 es la herramienta ideal para darle un toque personal y único a nuestro sitio web.

JS

Ahora, es necesario darle un poco de vida, y esto es posible gracias a JavaScript. Este lenguaje nos brinda la opción de controlar los eventos y hacer funcional nuestra web.

NodeJS

Si quieres una aplicación totalmente en tiempo real, robusta, actual, sin nada que envidiar a otras aplicaciones o sitios web, NodeJS es la herramienta ideal.

Django

Python es un lenguaje de alto nivel, y junto a él, el framework más popular; Django. Con este framework tendrás una aplicación totalmente modular y elegante.
